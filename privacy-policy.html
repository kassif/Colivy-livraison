Politique de Confidentialité de Colivy

Dernière mise à jour : 6 juin 2025

Cette politique explique comment Colivy ("nous", "notre application") traite vos données personnelles. En utilisant l'application, vous acceptez les termes décrits ci-dessous.

1. Données Collectées

Nous collectons :

  • Données de compte : Email, numéro de téléphone (via Firebase Authentication).
  • Localisation : Uniquement avec votre consentement, pour activer les fonctionnalités géolocalisées (livraisons, etc.).
  • Données d'usage : Interactions avec l'application (via Firebase Analytics).
  • Contenu utilisateur : Commandes, messages, profils (stockés dans Firestore).

2. Finalités du Traitement

Vos données servent à :

  • Exécuter vos commandes et livraisons.
  • Améliorer l'application et prévenir les abus.
  • Envoyer des notifications (si activées).
  • Répondre aux obligations légales.

3. Bases Légales

Le traitement repose sur :

  • Votre consentement (pour la localisation et les analytics).
  • L'exécution contractuelle (pour les commandes).
  • Nos intérêts légitimes (sécurité, amélioration de l'app).

4. Partage des Données

Vos données peuvent être transmises à :

  • Prestataires techniques : Firebase (Google), hébergeurs.
  • Partenaires de livraison (si applicable).
  • Autorités : Si requis par la loi.

Aucune vente à des tiers à des fins commerciales.

5. Sécurité

Nous utilisons :

  • Chiffrement des données (Firebase).
  • Contrôles d'accès stricts.

6. Durée de Conservation

Vos données sont conservées :

  • Jusqu'à suppression de votre compte (max. 1 an après inactivité).
  • Données facturation : 10 ans (obligation légale).

7. Vos Droits (RGPD)

Vous pouvez :

  • Accéder, rectifier ou supprimer vos données.
  • Retirer votre consentement (paramètres de l'app).
  • Portabilité des données (format lisible).
  • Déposer une réclamation auprès d'une autorité (CNIL).

Pour exercer ces droits, contactez-nous via : kassiferdi271@gmail.com.

8. Modifications

Les mises à jour seront notifiées dans l'application. La version actuelle est toujours disponible sur cette page.
