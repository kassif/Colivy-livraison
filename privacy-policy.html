Politique de Confidentialité

Dernière mise à jour : 6 juin 2025

Cette politique de confidentialité décrit comment l'application Colivy collecte, utilise et protège les informations personnelles de ses utilisateurs.

1. Informations collectées

L'application peut collecter les types de données suivants :

  • Données de localisation : utilisées uniquement pour afficher votre position sur la carte et permettre les livraisons ou services liés à votre position.
  • Données Firebase : telles que l’authentification (email, numéro de téléphone) et les informations stockées dans Firestore (commandes, profils, etc.).
  • Données analytiques : via Firebase Analytics, pour comprendre comment l’application est utilisée et l’améliorer.

2. Utilisation des données

Les données collectées sont utilisées pour :

  • Fournir les fonctionnalités principales de l'application
  • Améliorer l'expérience utilisateur
  • Assurer la sécurité des utilisateurs
  • Analyser les performances de l'application

3. Partage des données

Nous ne partageons vos données personnelles avec aucun tiers, sauf si cela est nécessaire pour fournir nos services (par exemple : Firebase, Google Maps).

4. Stockage et sécurité

Les données sont stockées de manière sécurisée sur Firebase (Google Cloud). L’application utilise des mécanismes de sécurité pour protéger vos informations.

5. Vos droits

Vous pouvez à tout moment demander la suppression de vos données ou contacter le développeur via le Play Store pour toute demande concernant la vie privée.

6. Modifications

Cette politique peut être mise à jour à l’avenir. Toute modification importante sera communiquée dans l’application.

7. Contact

Pour toute question concernant cette politique de confidentialité, veuillez nous contacter via la fiche de l'application sur le Google Play Store.
